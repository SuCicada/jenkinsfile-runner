# usage 
```bash
wget https://github.com/SuCicada/jenkinsfile-runner/releases/download/1.0-beta-30-with-jenkins/jenkinsfile-runner.tar.gz
tar zxvg jenkinsfile-runner.tar.gz
cd jenkinsfile-runner
./jenkinsfile-runner --help
```
